# 7 Континент — Система бронирования коттеджей

Веб-приложение для аренды и бронирования коттеджей в Дагестане. Проект включает в себя клиентскую часть для пользователей (просмотр домиков, бронирование, личный кабинет, отзывы) и мощную панель администратора для управления бронями и контентом.

## Технологический стек

**Фронтенд:**
* React 18 + TypeScript
* Vite (Сборщик)
* Zustand / Context API (Стейт-менеджмент)
* SCSS (Стилизация)

**Бэкенд:**
* Python 3 + Django 4
* Django REST Framework (DRF)
* PostgreSQL (База данных)

**Инфраструктура:**
* Docker & Docker Compose
* Nginx (Веб-сервер и обратный прокси)
* Let's Encrypt (SSL сертификаты)

## Структура проекта

* `/coastal-village1` — Фронтенд-приложение (React/Vite). Содержит свой Dockerfile и конфигурацию Nginx для статики.
* `/sevencontinent` — Бэкенд-приложение (Django/DRF). Включает приложения для аккаунтов, бронирований, отзывов и кастомную админку.
* `docker-compose.yml` — Конфигурация для одновременного запуска базы данных, бэкенда и фронтенда.
* `init-letsencrypt.sh` — Скрипт для автоматического получения SSL-сертификатов при деплое на боевой сервер.

## Локальный запуск (Development)

Для запуска проекта на вашем компьютере должны быть установлены [Docker](https://www.docker.com/) и [Docker Compose](https://docs.docker.com/compose/).
